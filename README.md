# PawTrace - Pet Tracking Web Application

A simple website to help people find their lost pets. Users can report missing pets, get alerts, track location, and offer rewards.

---

## What This Project Does

- Report a lost pet
- Get alerts about missing pets nearby
- Track pet location on a map
- Offer rewards for finding pets
- Submit feedback about the website

---

## How to Run

1. Download or copy the project folder
2. Open the folder
3. Double click on **INDEX.html**
4. The website opens in your browser

No installation needed. No internet needed (except for the map page).

---

## How to Use

**Report a Lost Pet**
- Go to the Report page
- Fill in your pet's name, type, and last location
- Click submit

**Alerts**
- Go to the Alert page
- You will see cards showing recently lost pets

**Tracking**
- Go to the Tracking page
- A map shows where pets were last seen

**Reward**
- Go to the Reward page
- See which pets have a reward offered
- Contact the owner if you find the pet

**Feedback**
- Go to the Feedback page
- Give a rating and write a comment
- Click submit
